# Privacy Policy

Last Updated: 2026-02-17

Knowledge Vault (“we”, “our”, “us”) explains how data is handled in the app.

## Data We Collect
- Account data: email, username
- Content data: notes, tasks, journal entries, tags, categories
- Device/app data: platform, app version, sync timestamps

## How We Use Data
- Provide core app functionality
- Enable authentication and sync
- Improve reliability and support

## Data Storage
- Local-first storage using on-device SQLite
- Optional cloud sync when enabled

## Data Security
- Sensitive credentials stored in secure storage where supported
- Encrypted transport for network communication
- Access controls and authentication checks on protected endpoints

## User Rights
- Access your data in-app
- Export your data
- Delete account/data (see Data Deletion Instructions)

## Third-Party Services
- MVP target: no advertising trackers
- Infrastructure vendors may process data solely to provide service

## Cookies and Tracking
- No advertising cookies or cross-site tracking for MVP

## Children's Privacy
- Intended for users age 13+
- Not directed to children under 13 (COPPA)
