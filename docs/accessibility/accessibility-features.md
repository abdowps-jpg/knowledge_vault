# Accessibility Features

## Current Focus Areas
- Semantic labels for interactive controls
- Keyboard accessibility on web
- Logical focus order
- Color contrast checks in light/dark themes

## WCAG 2.1 AA Checklist
- [ ] Buttons/inputs have labels and hints
- [ ] Focus visible and logical
- [ ] Contrast >= 4.5:1 (normal text)
- [ ] Supports font scaling
- [ ] Supports reduced motion

## Tooling
- Axe DevTools
- Lighthouse
- Android Accessibility Scanner
- VoiceOver / TalkBack manual testing
